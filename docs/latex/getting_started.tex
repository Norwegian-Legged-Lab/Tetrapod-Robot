The following introduces a short guide on how to get started with the source code for ASTRo.\hypertarget{getting_started_requirements}{}\doxysection{Requirements}\label{getting_started_requirements}
This library is written in C++.


\begin{DoxyItemize}
\item \href{http://wiki.ros.org/noetic}{\texttt{ ROS Noetic}}
\item \href{http://gazebosim.org/}{\texttt{ Gazebo 11.\+0.\+0}} (Usually comes as a part of the ROS installation)
\item \href{http://eigen.tuxfamily.org}{\texttt{ Eigen 3.\+2.\+0}}
\item \href{https://github.com/anybotics/kindr}{\texttt{ Kindr 1.\+0.\+0}} (Included in the catkin workspace\textquotesingle{}s source code, i.\+e., it can be considered a ROS package built with catkin)
\item \href{https://drake.mit.edu/}{\texttt{ Drake Toolbox}}
\item \href{https://ignitionrobotics.org/docs/citadel}{\texttt{ Ignition Citadel}} (Only experimental code depends on Ignition (the new simulation environment succeeding Gazebo))
\end{DoxyItemize}\hypertarget{getting_started_install}{}\doxysection{Installation}\label{getting_started_install}

\begin{DoxyCode}{0}

\DoxyCodeLine{//cd build}
\DoxyCodeLine{//cmake ..}
\DoxyCodeLine{//make}
\DoxyCodeLine{//}

\end{DoxyCode}


The library can additionally be installed by 
\begin{DoxyCode}{0}
\DoxyCodeLine{//sudo make install}
\DoxyCodeLine{//}

\end{DoxyCode}
 This will copy the file Find\+Kindr.\+cmake to the module path of CMake.\hypertarget{getting_started_build_lib_ros}{}\doxysubsection{Install the Library from Debian Package for ROS}\label{getting_started_build_lib_ros}

on the command line

\begin{DoxyItemize}

\begin{DoxyCode}{0}

\begin{DoxyCode}{0}

\DoxyCodeLine{//git clone git@github.com:anybotics/kindr.git}
\DoxyCodeLine{//catkin build -\/w \string~/catkin\_ws kindr}

Kindr can be included in your catkin project with\+: Add the following to your {\itshape CMake\+Lists.\+txt}\+:

\DoxyCodeLine{//}

\end{DoxyCode}

\begin{DoxyCode}{0}

\DoxyCodeLine{//mkdir build}
\DoxyCodeLine{//cd build}
\DoxyCodeLine{//cmake .. -\/DBUILD\_TEST=True}
\DoxyCodeLine{//make}
\DoxyCodeLine{//}
