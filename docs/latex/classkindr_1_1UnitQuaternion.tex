\subsubsection*{template$<$typename Prim\+Type\+\_\+$>$\newline
class kindr\+::\+Unit\+Quaternion$<$ Prim\+Type\+\_\+ $>$}

Implementation of a unit quaternion based on Eigen\+::\+Quaternion. 

The Hamiltonian convention is used, where Q = w + x$\ast$i + y$\ast$j + z$\ast$k and i$\ast$i=j$\ast$j=k$\ast$k=ijk=-\/1.

The following two typedefs are provided for convenience\+:
\begin{DoxyItemize}
\item Unit\+QuaternionF for float
\item Unit\+QuaternionD for double
\end{DoxyItemize}

\begin{DoxySeeAlso}{See also}
rm\+::\+Quaternion for an implementation of a generic quaternion 

rm\+::rotations\+::\+Rotation\+Quaternion for \mbox{\hyperlink{group__quaternions}{Quaternions}} that represent a rotation 
\end{DoxySeeAlso}
