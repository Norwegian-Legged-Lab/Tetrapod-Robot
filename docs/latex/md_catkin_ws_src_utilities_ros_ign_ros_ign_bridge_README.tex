In this example, we\textquotesingle{}re going to generate Ignition Transport images using Gazebo, that will be converted into R\+OS images, and visualized with {\ttfamily rqt\+\_\+viewer}.

\DoxyCodeLine{\# Shell A:}
\DoxyCodeLine{. /opt/ros/noetic/setup.bash}

\begin{DoxyCode}{0}
\DoxyCodeLine{\# Shell B:}
\DoxyCodeLine{gazebo}
\end{DoxyCode}


Once Gazebo is running, click on the {\ttfamily Insert} tab, and then, insert a {\ttfamily Camera} object into the scene. Now, let\textquotesingle{}s see the topic where the camera images are published.


\begin{DoxyCode}{0}
\DoxyCodeLine{\# Shell C:}
\DoxyCodeLine{ign topic -\/l | grep image}
\DoxyCodeLine{/default/camera/link/camera/image}
\end{DoxyCode}

\begin{DoxyCode}{0}
\DoxyCodeLine{\# Shell D:}

\begin{DoxyCode}{0}

\DoxyCodeLine{rqt\_image\_view /default/camera/link/camera/image}
\end{DoxyCode}
