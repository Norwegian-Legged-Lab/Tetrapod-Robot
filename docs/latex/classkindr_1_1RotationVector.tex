\doxysubsection{Detailed Description}
Implementation of a rotation vector based on Eigen\+::\+Matrix$<$\+Scalar, 3, 1$>$ 

The rotation vector is a non-\/normalized three-\/dimensional vector. The direction of the vector specifies the rotation axis and the length the angle. This representation uses therefore three parameters. \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{classkindr_1_1AngleAxis}{Angle\+Axis}} for an angle-\/axis representation with four parameters.
\end{DoxySeeAlso}
The following four typedefs are provided for convenience\+:
\begin{DoxyItemize}
\item Rotation\+Vector\+AD for primitive type double
\item Rotation\+Vector\+AF for primitive type float
\end{DoxyItemize}
