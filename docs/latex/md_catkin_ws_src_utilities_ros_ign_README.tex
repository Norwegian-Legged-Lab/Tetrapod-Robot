This repository holds packages that provide integration between \href{http://www.ros.org/}{\texttt{ R\+OS}} and \href{https://ignitionrobotics.org}{\texttt{ Ignition}}\+:


\begin{DoxyItemize}
\item \href{https://github.com/osrf/ros_ign/tree/noetic/ros_ign}{\texttt{ ros\+\_\+ign}}\+: Metapackage which provides all the other packages.
\item \href{https://github.com/osrf/ros_ign/tree/noetic/ros_ign_image}{\texttt{ ros\+\_\+ign\+\_\+image}}\+: Unidirectional transport bridge for images from \href{https://ignitionrobotics.org/libs/transport}{\texttt{ Ignition Transport}} to R\+OS using \href{http://wiki.ros.org/image_transport}{\texttt{ image\+\_\+transport}}.
\item \href{https://github.com/osrf/ros_ign/tree/noetic/ros_ign_bridge}{\texttt{ ros\+\_\+ign\+\_\+bridge}}\+: Bidirectional transport bridge between \href{https://ignitionrobotics.org/libs/transport}{\texttt{ Ignition Transport}} and R\+OS.
\item \href{https://github.com/osrf/ros_ign/tree/noetic/ros_ign_gazebo}{\texttt{ ros\+\_\+ign\+\_\+gazebo}}\+: Convenient launch files and executables for using \href{https://ignitionrobotics.org/libs/gazebo}{\texttt{ Ignition Gazebo}} with R\+OS.
\item \href{https://github.com/osrf/ros_ign/tree/noetic/ros_ign_gazebo_demos}{\texttt{ ros\+\_\+ign\+\_\+gazebo\+\_\+demos}}\+: Demos using the R\+O\+S-\/\+Ignition integration.
\item \href{https://github.com/osrf/ros_ign/tree/noetic/ros_ign_point_cloud}{\texttt{ ros\+\_\+ign\+\_\+point\+\_\+cloud}}\+: Plugins for publishing point clouds to R\+OS from \href{https://ignitionrobotics.org/libs/gazebo}{\texttt{ Ignition Gazebo}} simulations.
