\doxysubsection{Detailed Description}
System which publishes R\+OS Point\+Cloud2 messages for R\+G\+BD or G\+PU lidar sensors. 

This plugin should be attached to an R\+G\+BD or G\+PU lidar sensor (i.\+e. $<$sensor...$>$$<$plugin$>$)

Important\+: load {\ttfamily ignition\+::gazebo\+::systems\+::\+Sensors} as well, which will create the sensor.

\item {\ttfamily $<$namespace$>$}\+: Namespace for R\+OS node, defaults to sensor scoped name
\item {\ttfamily $<$topic$>$}\+: R\+OS topic to publish to, defaults to \char`\"{}points\char`\"{}
\item {\ttfamily $<$frame\+\_\+id$>$}\+: TF frame name to populate message header, defaults to sensor scoped name
\item {\ttfamily $<$engine$>$}\+: Render engine name, defaults to \textquotesingle{}ogre2\textquotesingle{}
\item {\ttfamily $<$scene$>$}\+: Scene name, defaults to \textquotesingle{}scene\textquotesingle{} 
\end{DoxyItemize}
