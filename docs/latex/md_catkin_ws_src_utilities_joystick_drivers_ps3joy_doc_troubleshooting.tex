When pairing your joystick via bluetooth, you may recieve the following message on your terminal\+: ~\newline
 
\begin{DoxyCode}{0}
\DoxyCodeLine{Current Bluetooth master: 00:15:83:ed:3f:21}
\DoxyCodeLine{Unable to retrieve local bd\_addr from `hcitool dev`.}
\DoxyCodeLine{Please enable Bluetooth or specify an address manually.}
\end{DoxyCode}


This would indicate that your bluetooth is disabled. To enable your bluetooth, try the following\+:


\begin{DoxyEnumerate}
\item Check the status of your bluetooth by entering the following\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{sudo systemctl status bluetooth}
\end{DoxyCode}

\end{DoxyEnumerate}

You may see something like this\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{● bluetooth.service -\/ Bluetooth service}
\DoxyCodeLine{   Loaded: loaded (/lib/systemd/system/bluetooth.service; disabled; vendor preset: enabled)}
\DoxyCodeLine{   Active: inactive (dead)}
\DoxyCodeLine{     Docs: man:bluetoothd(8)}
\end{DoxyCode}


If you do, that means your bluetooth service is disabled. Turn enable it enter 
\begin{DoxyCode}{0}
\DoxyCodeLine{sudo systemctl start bluetooth}
\DoxyCodeLine{sudo systemctl status bluetooth  }
\end{DoxyCode}


After running these commands your bluetooth service should be up and running\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{● bluetooth.service -\/ Bluetooth service}
\DoxyCodeLine{   Loaded: loaded (/lib/systemd/system/bluetooth.service; disabled; vendor preset: enabled)}
\DoxyCodeLine{   Active: active (running) since Thu 2017-\/06-\/29 16:21:43 EDT; 16s ago}
\DoxyCodeLine{     Docs: man:bluetoothd(8)}
\DoxyCodeLine{ Main PID: 27362 (bluetoothd)}
\DoxyCodeLine{   Status: "Running"}
\DoxyCodeLine{   CGroup: /system.slice/bluetooth.service}
\DoxyCodeLine{           └─27362 /usr/local/libexec/bluetooth/bluetoothd}
\end{DoxyCode}


Retry the commands that were mentioned in step 2 for pairing the P\+S3 joystick.


\begin{DoxyEnumerate}
\item Run the following command\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{hciconfig hci0 reset }
\end{DoxyCode}

\end{DoxyEnumerate}

followed by\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{sudo bash }
\DoxyCodeLine{rosrun ps3joy sixpair}
\end{DoxyCode}
